This wordpress plugin stores all xAPI statements generaded by H5P in a remote LRS.

== Description ==
wp-h5p-xapi interfaces between the h5p wordpress plugin and Learning Record Stores (LRSes) such as Learning locker(http://learninglocker.net/). It auto saves xapi statements generated by some h5p libraries into an LRS.

How to use
Once your installation is complete, go to Settings >> H5P xAPI. Enter the credentials for the connection to the LRS and you are good to go!

== Installation ==
Download the plugin and upload it in the plugins section of your wordpress instance.

Optionally, you can do the following.
1) Download and extract the plugin
2) Put the extracted folder in your `/wp-content/plugins/` directory
3) Navigate to your wordpress instance
4) Login and active the plugin
